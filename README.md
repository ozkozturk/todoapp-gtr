# To Do List App

## Install

`yarn install`

## Start

`yarn start`

## Build

`yarn build`

## Versions

`node: v14.10.1`

`yarn: 1.22.5`

## Techs

- [React](https://en.reactjs.org/)
- [Redux](https://redux.js.org/)
- [Redux Thunk](https://github.com/reduxjs/redux-thunk)
- [TypeScript](https://www.typescriptlang.org/)
- [emotion](https://emotion.sh/)
- [Tabler Icons](https://tablericons.com/)
